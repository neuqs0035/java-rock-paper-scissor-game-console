## How to Run

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/neuqs0035/java-rock-paper-scissor-game-console.git
   cd java-rock-paper-scissor-game-console
   ```
2. **Compile Java File**
   ```bash
   javac Main.java
   ```

3. **Run Java File:**
   ```bash
   java Main
   ```
